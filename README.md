## Philosophy

Most dark themes use cold, sterile grays that feel like working in a server room. Envying Baby is different. It uses **warm chocolates** with brown undertones that feel like a cozy cabin at night, and **luminous coral accents** that glow like a campfire in the darkness.

This theme is designed for those 14-hour debugging sessions when your eyes need kindness and your heart needs... well, maybe we don't talk about that.

## Customization

This theme supports the [Style Settings](https://github.com/mgmeyers/obsidian-style-settings) plugin. Install it to access these options:

**Colors**
- Accent Color (light mode)
- Accent Color (dark mode)

**Typography**
- Editor Font
- Code Font
- Line Height

**Effects**
- Enable/Disable Glow Effects
- Enable/Disable Animations

## Special Callouts

This theme includes special callout types that match the aesthetic:

```markdown
> [!love] Valentine's Note
> For your romantic debugging sessions.

> [!heart] From the Heart
> When code speaks to the soul.

> [!valentine] Special Message
> Because every commit deserves love.
```

These use the coral accent color with full glow effects in dark mode.
